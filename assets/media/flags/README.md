# Drapeaux pour le sélecteur de langue

Ce dossier doit contenir les images des drapeaux pour le sélecteur de langue.

## Images requises:

- `france.png` - Drapeau français (20x14px ou proportionnel)
- `uk.png` - Drapeau anglais (20x14px ou proportionnel)
- `spain.png` - Drapeau espagnol (20x14px ou proportionnel)
- `germany.png` - Drapeau allemand (20x14px ou proportionnel)

## Sources recommandées:

Vous pouvez télécharger des drapeaux gratuits depuis:
- https://flagicons.lipis.dev/ (Flag Icons)
- https://www.flaticon.com/packs/country-flags
- https://www.iconfinder.com/search?q=flag&price=free

## Format recommandé:
- Format: PNG avec transparence
- Taille: 20x14px (ou dimensions proportionnelles)
- Optimisé pour le web

## Utilisation temporaire:

Si vous n'avez pas encore les images, vous pouvez utiliser des emojis ou du texte en attendant en modifiant le HTML dans index.html.
